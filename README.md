## 🎯 О проекте

Animedia-Onlite-clone — это одностраничное приложение (SPA), которое воспроизводит функциональность и дизайн популярного сайта с аниме контентом. Проект демонстрирует возможности React для создания сложных пользовательских интерфейсов с системой маршрутизации, авторизацией и адаптивным дизайном.

## 🧭 Навигация и маршруты

Приложение использует **React Router v6** для навигации. Вот все доступные маршруты:

| Маршрут | Компонент | Макет | Описание |
|---------|-----------|-------|----------|
| `/` | `<Home/>` | `<Layout>` | **Главная страница** — лента с карточками аниме, слайдер, "аниме дня" |
| `/anime/:id` | `<AnimePage/>` | `<Layout>` | **Страница аниме** — детальная информация об аниме, таймер выхода новых серий, комментарии |
| `/registration` | `<Register/>` | `<Layout>` | **Регистрация** — форма создания нового аккаунта |
| `/login` | `<Login/>` | `<Layout>` | **Вход** — форма авторизации пользователя |
| `/profile` | `<UserProfile/>` | `<LayoutProfile>` | **Профиль пользователя** — личный кабинет (использует отдельный макет) |
| `/search` | `<SearchPage/>` | `<Layout>` | **Поиск** — страница поиска аниме |
| `/about` | `<Story/>` | `<Layout>` | **О нас** — информационная страница, история проекта |
| `/products` | `<Products/>` | `<Layout>` | **Товары** — страница с мерчем/продукцией |

### 🏗️ Макеты (Layouts)

В проекте используется два типа макетов:

- **`<Layout>`** — основной макет для большинства страниц (содержит header, footer, основную структуру)
- **`<LayoutProfile>`** — специальный макет для профиля пользователя (возможно, с другой навигацией или боковым меню)

## ✨ Функциональность

- ✅ **Маршрутизация** — плавная навигация между страницами без перезагрузки
- ✅ **Аутентификация пользователей** — регистрация (`/registration`), вход (`/login`), профиль (`/profile`)
- ✅ **Главная страница** (`/`) — лента с карточками аниме, слайдер, "аниме дня"
- ✅ **Детальная страница аниме** (`/anime/:id`) — полная информация, таймер выхода новых серий, комментарии
- ✅ **Поиск** (`/search`) — возможность поиска аниме
- ✅ **Информационные страницы** — "О нас" (`/about`), магазин (`/products`)
- ✅ **Адаптивный дизайн** — корректное отображение на всех устройствах

## 🛠️ Технологии

- **Frontend**: React 18
- **Маршрутизация**: React Router v6
- **Стилизация**: CSS/SCSS модули
- **HTTP-клиенты**: для работы с API
- **Утилиты**: работа с датами, токенами авторизации
- **Инструменты**: Create React App, Webpack, Babel
